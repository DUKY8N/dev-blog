# Dev Blog

Personal development blog built with [Astro](https://astro.build/).

## Getting Started

```sh
npm install
npm run dev
```

## Commands

| Command | Description |
| --- | --- |
| `npm run dev` | Start the local development server |
| `npm run build` | Build the site to `dist/` |
| `npm run preview` | Preview the production build locally |

## Content

Blog posts live in `blog/` and are grouped by section:

```text
blog/
├── notes/
├── logs/
└── thoughts/
```

Use kebab-case slugs for filenames.
